# 📚 Mango - Online Book Borrowing Platform

Mango is a modern online book borrowing platform built with Next.js, Better Auth, and MongoDB. It digitizes the traditional library experience by allowing users to browse books, search by title, filter by category, and securely manage their accounts.

## 🚀 Features

- Secure authentication using Better Auth
- Email/Password and Google Sign-In
- Browse all available books
- Search books by title
- Filter books by category (Story, Tech, Science)
- Featured Books section on the homepage
- Private Book Details page
- User Profile page
- Update profile information
- Responsive design for Mobile, Tablet, and Desktop
- Modern and clean UI with smooth animations

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/tawsifulMostafa/assignment_no_008.git
```

Go to the project folder

```bash
cd your-repository
```

Install dependencies

```bash
npm install
```

Create a `.env.local` file and add your environment variables.

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```

## 📁 Environment Variables

Create a `.env.local` file and add the required environment variables.

```env
BETTER_AUTH_SECRET=
BETTER_AUTH_URL=
MONGODB_URI=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
```
